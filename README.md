# NORDHEM — sleep, live, store

A modern Nordic home e-commerce storefront with a real search-engineering brain: Elasticsearch-powered product search (query understanding, typo tolerance, synonyms, facets, semantic/hybrid ranking), a Search Studio with offline relevance evaluation on 233k human judgments, editor tooling, analytics, and a graceful-degradation failover tier.
